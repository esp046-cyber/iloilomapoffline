- Southern coast along Guimaras Strait
- Northeastern "panhandle" region
- Inland mountainous areas

### Municipality Positioning
- **Coastal municipalities** positioned along actual coastlines

- **Adjacent provinces**: Properly positioned around Iloilo

## Usage Instructions

1. **Open the Application**: Load `iloilo_map_realistic.html` in any web browser
2. **Explore the Map**: 
   - Click on any municipality or city to view detailed information
   - Use the search box to find specific LGUs
   - Hover over areas for quick information
3. **Mobile Use**: Optimized for touch devices with appropriate scaling
4. **Offline Operation**: No internet connection required
